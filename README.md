# -codealpha_tasks# 🎓 Student Grade Tracker

A Java console-based application developed for the **CodeAlpha Java Programming Internship**. This project helps manage student records, track grades, and generate detailed performance reports using Object-Oriented Programming concepts and Java Collections.

## 📌 Features

* ➕ Add new students
* 📝 Add multiple grades for each student
* 👀 View individual student grades
* 📊 Calculate average scores
* 🏆 Find highest and lowest grades
* 🎯 Generate letter grades (A, B, C, D, F)
* ❌ Remove student records
* 📈 Display class performance summary
* 📋 Generate grade distribution reports
* ⚠️ Input validation and exception handling
* 🖥️ Interactive menu-driven console interface

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Collections Framework
* Exception Handling
* Scanner Class
